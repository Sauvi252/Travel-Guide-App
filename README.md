## 📸 App Overview

**Smart India Guide** is a comprehensive travel companion app that helps users discover, plan, and experience India's incredible tourist destinations. From the snowy peaks of Uttarakhand to the beaches of Goa, the app covers all major states with detailed place information, AI-powered itinerary generation, AR-based landmark discovery, and a digital travel passport system.

---

## ✨ Features

### 🗺️ Explore India
- Browse **26+ tourist destinations** across **21 Indian states**
- Beautiful **state grid view** with real images for each state
- Detailed place cards with photos, history, timings, entry fees, and travel tips
- **Search** places by name, location, or state
- **Voice search** support for hands-free exploration
- **State filter chips** for quick filtering by region

### 🤖 AI Trip Planner (Gemini Powered)
- Enter destination, number of days, and budget
- Get a **fully customized day-by-day itinerary** generated by Google Gemini AI
- Morning, afternoon, and evening activity breakdowns
- Local food recommendations and travel tips included

### 🥽 AR Guide (ARCore)
- Point your camera at monuments to **detect landmarks** in real time
- Overlays historical information and details directly on screen
- Powered by **Google ARCore**

### 🌦️ Weather Integration
- Live weather display on the home screen
- Shows current temperature and conditions for popular destinations

### 🗺️ Maps & Nearby Places
- **View on Google Maps** with one tap
- Search for **hotels** and **restaurants** near any destination
- Deep links into Google Maps for directions

### 👥 Travel Buddy Finder
- Connect with other travelers visiting the same place
- Find solo travelers, group travelers, and food tour buddies
- Social discovery feature for shared experiences

### 🎫 Digital Travel Passport
- Track places visited, states explored, and badges earned
- Gamified achievement system with unlockable badges (Taj Explorer, Forest King, etc.)
- View stats: places visited, states covered, badges collected
- Share your passport to Instagram

### 💰 Expense Tracker
- Track travel expenses across trips
- View total spending at a glance
- Add and categorize travel expenses

### 👤 User Profile
- Login and registration with form validation
- Firebase Authentication integration (ready for production)
- Profile page with quick access to all features

### 🌱 Eco & Safety Scores
- **Sustainable Travel Score** displayed per destination (e.g., 92% Eco-Friendly)
- **Safety Score** badge shown on each place detail page
- AI Price Insights panel with booking predictions

## 🚀 Getting Started

### Prerequisites
- Android Studio Hedgehog or later
- Android device / emulator running **API 24+**
- Google account for Firebase setup
- Gemini API key (for AI Trip Planner)
- ARCore-compatible device (for AR Guide)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/SauravKumar/Travel-Guide-App.git
   cd Travel-Guide-App
   ```

2. **Open in Android Studio**
   - File → Open → select the project folder

3. **Firebase Setup**
   - Go to [Firebase Console](https://console.firebase.google.com/)
   - Create a new project
   - Add an Android app with package name `com.example.travelguideapp`
   - Download `google-services.json` and place it in the `app/` directory

4. **Gemini API Key**
   - Get your API key from [Google AI Studio](https://aistudio.google.com/)
   - Open `TripPlannerFragment.java`
   - Replace `"PASTE_YOUR_API_KEY"` with your actual key
   ```java
   private static final String GEMINI_API_KEY = "your_actual_api_key_here";
   ```
   > ⚠️ **Warning:** Never commit API keys to version control. Use environment variables or a backend proxy in production.

5. **Build & Run**
   - Connect your Android device or start an emulator
   - Click **Run** ▶ in Android Studio
